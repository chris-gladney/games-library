# Games Library

## Overview
Welcome to my games library! This repository contains a library of games built using different tech stacks. Each game will have its own section in this README document. Enjoy!

### Draughts
This is the game of draughts (or checkers). Played on an 8x8 grid, with a white team and a black team. White moving first, the aim is for other player to either have no legal moves or no pieces left on the board. This is a two player game (no single player functionality).
